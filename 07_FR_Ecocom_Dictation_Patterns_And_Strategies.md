# Dictation Patterns & Strategies (Numbers, Letters, Codes)

Helps with tasks where numbers, letters and codes appear in written form.

---

## 1. Written French Numbers → Digits

When the task gives numbers in words, convert like this.

### Basic examples

- vingt-trois → 23  
- quatorze → 14  
- trente et un → 31  
- soixante-douze → 72  
- cinquante-neuf → 59  
- dix-sept → 17  
- quatre-vingt-un → 81  
- nonante-deux → 92  

### Strategy

1. See if the form is **Swiss** (*septante, nonante*) or **French** (*soixante-dix, quatre-vingt-dix*).  
2. Convert to digits.  
3. Keep the same grouping if used in phone numbers.

---

## 2. Digits → Written French Numbers

If asked to write numbers in words:

- 14 → quatorze  
- 40 → quarante  
- 16 → seize  
- 60 → soixante  
- 75 → soixante-quinze / septante-cinq  
- 98 → quatre-vingt-dix-huit / nonante-huit  

Stick to one variant (Swiss or French) inside the same answer.

---

## 3. Phone Numbers – Structure

Phone numbers often:

- 021 331 08 20  
- 027 771 80 98  
- 079 555 23 10  

Strategy:

- Write digits in groups of 2 or 3, as in the exercise.  
- If you hear a number, write it directly in digits; do not translate into words.

---

## 4. Codes with Letters and Signs

Example pattern:

- Spoken: « A majuscule tiret vingt-sept tiret bas dix-neuf barre oblique trois »  
- Written: `A-27_19/3`

Mapping reminders:

- tiret → -  
- tiret bas / underscore → _  
- barre oblique → /  
- point → .  
- arobase → @  

---

## 5. Email Addresses

Example:

- Spoken: « christine point pierre arobase mediafri point ch »  
- Written: `christine.pierre@mediafri.ch`

Pattern:

- `[nom] point [prénom] arobase [domaine] point [extension]`

Write exactly the dots and @ in the right place.

---

## 6. Strategy Hints for Practice

1. **Write group by group**

   - After each group (e.g. two digits), pause mentally before the next group.

2. **Separate letters and numbers**

   - Use capitals where needed (A, B, C).  

3. **Watch danger pairs**

   - 14 / 40 → quatorze / quarante  
   - 15 / 50 → quinze / cinquante  
   - 16 / 60 → seize / soixante  

4. **Check final result**

   - Quick re-scan to see if the number of digits / symbols matches what you heard / read.

---

## 7. Likely Exam Patterns

You can expect:

- Dictation of several **pure numbers** (2–3 digits each).  
- Dictation of **mixed codes** (letters + digits + signs).  
- Dictation of at least one **email address**.

Use the mappings in this file to reconstruct the written form exactly.
