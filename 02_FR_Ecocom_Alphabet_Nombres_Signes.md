# Alphabet, Numbers & Special Signs

This file groups all basic elements that appear in codes, phone numbers, emails and dictations.

---

## 1. Alphabet français (épeler)

Letter names in French:

- A → a  
- B → bé  
- C → cé  
- D → dé  
- E → e  
- F → effe  
- G → gé  
- H → ache  
- I → i  
- J → ji  
- K → ka  
- L → elle  
- M → emme  
- N → enne  
- O → o  
- P → pé  
- Q → qu  
- R → erre  
- S → esse  
- T → té  
- U → u  
- V → vé  
- W → double vé / double v  
- X → ixe  
- Y → i grec  
- Z → zède  

Only the sound matters for dictation tasks.

## 2. Accents et signes sur les lettres

- é → *e accent aigu*  
- è → *e accent grave*  
- ê → *e accent circonflexe*  
- ë → *e tréma*  
- à → *a accent grave*  
- â → *a accent circonflexe*  
- î → *i accent circonflexe*  
- ô → *o accent circonflexe*  
- ù → *u accent grave*  
- ç → *c cédille*  

Important especially in names and emails.

## 3. Signes de ponctuation et signes particuliers

Used in codes, email addresses, websites:

- . → **point**  
- , → **virgule**  
- : → **deux-points**  
- ; → **point-virgule**  
- ? → **point d’interrogation**  
- ! → **point d’exclamation**  
- - → **tiret / trait d’union**  
- _ → **tiret bas / underscore**  
- / → **barre oblique / slash**  
- ( ) → **parenthèses**  
- « » → **guillemets**  
- [ ] → **crochets**  
- * → **astérisque**  
- @ → **arobase**  
- # → **dièse**

## 5. Numéros de téléphone – Principes

### Lecture générale

- Numéros souvent lus **par groupes de 2 ou 3 chiffres**.  
- Exemples:  
  - 021 331 08 20  
  - 027 771 80 98  
  - 079 555 23 10  

### Indicatifs utiles (Suisse)

- 021 → Vaud  
- 022 → Genève  
- 026 → Fribourg  
- 027 → Valais  
- 032 → Neuchâtel / Jura / Jura bernois

## 6. Exemples de codes

Typical exam-style codes:

- Adresse e-mail:  
  - `christine.pierre@mediafri.ch`  
  - → “christine point pierre arobase mediafri point ch”

- Code client:  
  - `A-27_19/3`  
  - → “A tiret vingt-sept tiret bas dix-neuf barre oblique trois”

Use these patterns to decode or encode any similar code in tasks.
